# Инструкция по проекту Serial Controller LED

## 🎮 WinForms приложение

В папке `SerialControllerLed` находится код desktop-приложения на C# WinForms. Оно позволяет:
- Подключаться к COM-порту
- Управлять красным и синим светодиодами
- Обновлять список доступных портов

## 🤖 Код для Arduino

В файле `lamp_control.ino` находится код для Arduino, который:
- Принимает команды через Serial порт
- Управляет светодиодами согласно полученным командам

## 🔌 Подключение светодиодов к Arduino

| Цвет | Пин Arduino | Резистор |
|------|-------------|----------|
| 🔴 Красный LED | **Pin 12** | 220 Ом |
| 🔵 Синий LED | **Pin 8** | 220 Ом |

### Команды, которые отправляет приложение

| Команда | Действие |
|---------|----------|
| `ONRED\n`  | Включить красный LED |
| `OFFRED\n` | Выключить красный LED |
| `ONBLUE\n` | Включить синий LED |
| `OFFBLUE\n`| Выключить синий LED |

## 🚀 Запуск проекта

1. Загрузите скетч `lamp_control.ino` в Arduino
2. Подключите светодиоды согласно таблице (Pin 12 и Pin 8)
3. Запустите WinForms приложение из папки `SerialControllerLed`
4. Выберите COM-порт Arduino в выпадающем списке
5. Нажмите "Подключиться"
6. Управляйте светодиодами кнопками ON/OFF

## 📝 Примечания

- Убедитесь, что Arduino подключен к правильному COM-порту
- Скорость передачи данных: **9600 бод**
- Код предназначен только для обучения и экспериментов
